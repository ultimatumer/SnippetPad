# SnippetPad

`SnippetPad` — lightweight desktop utility для глобальных текстовых сниппетов: назначаешь хоткей, получаешь мгновенную подстановку подготовленного текста из любой программы.

## Что делает

- держит набор сниппетов в локальном хранилище;
- привязывает каждый сниппет к глобальному shortcut'у;
- позволяет быстро править, добавлять и удалять бинды из UI;
- поддерживает Unicode-текст, включая emoji, и многострочные шаблоны с переносами строк;
- работает из трея и не требует активного окна;
- подтягивает новые релизы без ручной переустановки.

## Почему это сильный инструмент

- Убирает рутину на повторяющемся вводе: шаблоны ответов, команды, промпты, куски документации, часто используемые блоки текста.
- Подходит не только для коротких фраз: можно хранить полноценные многострочные заготовки, оформленные ответы и тексты с emoji без отдельной обработки.
- Даёт глобальный слой shortcuts поверх всей системы, а не только внутри одного редактора или мессенджера.
- Хранит данные локально и не тащит лишнюю инфраструктуру: без аккаунтов, без облака, без сервера.
- Архитектурно лёгкий стек: Tauri + React на UI, Rust на системной части, поэтому приложение компактное и дешёвое по ресурсам.
- Трей-режим делает его ближе к utility-классу вроде launcher/expander, а не к обычному “окну с формой”.
- Встроенное автообновление сделано нормально: клиент проверяет релизы с GitHub, а обновления валидируются по публичному ключу подписи. Это уже не “скачай новый exe вручную”, а полноценный signed update pipeline.
